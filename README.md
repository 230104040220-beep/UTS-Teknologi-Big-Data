# Deskripsi Project

Project ini merupakan implementasi Big Data Pipeline menggunakan:

- PySpark
- Parquet
- Machine Learning (Linear Regression)
- Streamlit Dashboard
- Plotly Visualization

Sistem digunakan untuk menganalisis konsumsi energi dari beberapa sektor industri berbasis data sensor IoT.

---

# Pipeline System

Energy Sensor → Spark ETL → Parquet Warehouse → AI Forecasting → Dashboard
